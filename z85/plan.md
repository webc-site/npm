# Z85 编解码器设计方案与性能测试报告

## 1. 技术选型

- **开发语言**: 纯现代 JavaScript (ES6+)，完全浏览器兼容，不依赖 Node.js 特有 API (如 `Buffer`)。
- **性能测试工具**: `mitata` (Bun 原生级别极速 benchmark 库)。
- **表格化输出工具**: `cli-table3` (无边框样式配置，输出纯净的文本对齐表格)。
- **测试对照组/竞品**:
  - `base85`
  - `@lapo/z85`
  - `z85-codec`

## 2. 算法设计与 ACM 大师级优化

Z85 将 4 字节二进制编码为 5 字节 ASCII 字符。

### 优化核心逻辑与尝试历程：

我们在重构中对算法及数据结构进行了优化与验证：

1. **公共字符集 UTF-8 编码**：
   - 提取 Z85 标准字符集至 `src/_.js`，预先利用 `@3-/utf8` 的 `utf8e` 进行 UTF-8 编码，供编码和解码共享。避免字符串在两个模块中重复定义及运行时的重复编码开销。

2. **编码器设计 (`z85e.js`)**：
   - 保持极简化与无多余预计算的轻量化设计（minified 后仅 193 字节）。直接通过循环除以 85 / 取模 85 提取字符并利用 `ENCODE_MAP` 查表。

3. **解码器双层查表优化 (`z85d.js`)**：
   - 第一层 `DECODE_MAP`（256 字节 `Int8Array`）验证单字节合法性。
   - 第二层 `DECODE_MAP_2`（65536 字节 `Int16Array`）预计算所有可能的双字节组合（85×85=7225）对应的数值，将五字节解码分解为三次查表与两次乘加运算，从而避免频繁的循环乘除法开销，解码速度突破 **`1200 MB/s`**！

## 3. 性能测试结果 (基于吞吐量 MB/s 表格 `bench/run.js`)

运行环境：Apple M2 Max (runtime: bun 1.3.14)
_注：MB/s 为处理的二进制源数据吞吐率_

### 3.1 编码性能对比 (Encoding)

| 实现/库名称            | 大数据 (1MB) 吞吐量 | 小文本 (32B) 吞吐量 | 编码吞吐性能对比  |
| :--------------------- | :------------------ | :------------------ | :---------------- |
| **我们的实现 (`our`)** | **520.88 MB/s**     | **747.89 MB/s**     | **基准 (最快)**   |
| `codec` (z85-codec)    | 53.58 MB/s          | 88.12 MB/s          | 慢 9.7 ~ 8.4 倍   |
| `lapo` (@lapo/z85)     | 45.47 MB/s          | 61.61 MB/s          | 慢 11.4 ~ 12.1 倍 |
| `base85`               | 25.47 MB/s          | 26.09 MB/s          | 慢 20.4 ~ 28.6 倍 |

### 3.2 解码性能对比 (Decoding)

| 实现/库名称            | 大数据 (1MB) 吞吐量 | 小文本 (40B，解出 32B) 吞吐量 | 解码吞吐性能对比    |
| :--------------------- | :------------------ | :---------------------------- | :------------------ |
| **我们的实现 (`our`)** | **1296.59 MB/s**    | **973.00 MB/s**               | **基准 (最快)**     |
| `base85`               | 136.78 MB/s         | 53.23 MB/s                    | 慢 9.4 ~ 18.2 倍    |
| `lapo` (@lapo/z85)     | 19.44 MB/s          | 18.39 MB/s                    | 慢 66.6 ~ 52.9 倍   |
| `codec` (z85-codec)    | 3.08 MB/s           | 3.58 MB/s                     | 慢 420.9 ~ 271.7 倍 |
